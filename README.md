# Math Quiz Game — C++

Project built as part of the **Foundations of Programming** roadmap by Dr. Mohammed Abu-Hadhoud (Programming Advices).

## About the Project

This is a console-based Math Quiz Game built in C++. The game generates randomized math questions based on the user's selected difficulty and operation type, tracks correct and incorrect answers, and allows replaying multiple quiz sessions.

## Concepts Used

- Enums and structs
- Random number generation
- Input validation
- Functions and function decomposition
- String manipulation
- Loops and conditional logic
- Scoring and replay flow
